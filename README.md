## 🛠 技术栈

- **前端**: HTML5 · CSS3 · Vanilla JavaScript
- **部署**: GitHub Pages
- **数据**: GitHub Actions 定时同步 GitHub 统计数据

## 📁 目录结构

```
├── index.html              # 主页
├── 404.html                # 404 页面
├── css/                    # 样式表
│   ├── variables.css       # CSS 变量
│   ├── reset.css           # 重置样式
│   ├── layout.css          # 布局
│   ├── components.css      # 组件
│   ├── animations.css      # 动画
│   ├── article.css         # 文章页
│   ├── responsive.css      # 响应式
│   └── style.css           # 主样式（导入汇总）
├── js/                     # JavaScript
│   ├── main.js             # 入口
│   ├── theme.js            # 深色模式
│   ├── github.js           # GitHub 数据
│   ├── projects.js         # 项目列表
│   └── blogs.js            # 博客列表
├── blog/                   # 博客文章
├── projects/               # 项目详情页
├── data/                   # JSON 数据
│   ├── blogs.json
│   ├── projects.json
│   └── github-stats.json   # GitHub Actions 自动更新
└── images/                 # 图片资源
```

## 🚀 本地开发

直接用浏览器打开 `index.html` 即可，无需构建工具。
