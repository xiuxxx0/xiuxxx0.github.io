📚 软件设计师备考记录

发布时间：2026-07-17

为什么准备软件设计师考试

作为计算机科学与技术专业学生， 除了学习 Java 后端开发之外， 我也希望加强自己的计算机基础能力。 软件设计师考试覆盖了计算机组成原理、 数据结构、操作系统、数据库以及软件工程等内容， 这些知识对于理解软件开发非常重要。

学习内容

  • 💻 计算机组成原理
  • 📚 数据结构与算法
  • ⚙ 操作系统
  • 🗄 数据库系统
  • 🌐 网络基础
  • 📐 软件工程
  • ☕ 面向对象程序设计

计算机组成原理学习记录

目前重点学习计算机组成原理部分。 主要内容包括：

  • 二进制与数制转换
  • 原码、反码、补码
  • 移码表示
  • CPU结构与指令系统
  • Cache、主存、辅存组成的存储系统
  • 海明码与CRC校验

数据结构学习记录

数据结构部分主要复习：

  • 线性表
  • 栈和队列
  • 树和二叉树
  • 图结构
  • 排序算法
  • 查找算法

学习方法

目前采用：

  • 理解知识点，而不是单纯背答案
  • 通过刷题巩固计算机基础
  • 整理错题和重点知识
  • 结合 Java 编程理解理论知识

目前遇到的问题

软件设计师涉及知识范围较广， 刚开始学习时容易感觉内容零散。 例如计算机组成原理中的补码、 移码、存储系统等内容， 需要不断结合例题理解。

通过持续学习， 逐渐建立起计算机系统整体认识。

未来计划

接下来计划继续学习：

  • 数据结构算法强化
  • 操作系统重点知识
  • 数据库设计
  • 软件设计师案例分析

总结

软件设计师备考不仅是为了考试， 更重要的是帮助自己建立计算机专业基础。 未来希望能够将这些基础知识应用到 Java 后端开发和实际项目中。
