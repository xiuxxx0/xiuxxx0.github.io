☕ Java学习第一周：从 Hello World 到面向对象

发布时间：2026年7月

学习背景

作为一名计算机科学与技术专业学生， 我开始系统学习 Java 后端开发。 相比单纯学习语法， 我希望通过实际项目理解 Java 在软件开发中的应用。

第一阶段：认识Java

第一周主要学习 Java 基础环境搭建， 包括 JDK 安装、IDE 配置以及第一个 Java 程序。 


public class Hello {

    public static void main(String[] args){

        System.out.println("Hello Java");

    }

}

通过 Hello World 程序， 我第一次理解了 Java 程序的基本结构： 类、方法以及 main 方法入口。

学习内容

  • 变量与数据类型
  • Scanner输入输出
  • if条件判断
  • for、while循环
  • 数组基础
  • 面向对象基础

第一个小项目

为了巩固基础， 我完成了一个简单的猜数字小游戏。

  • 随机生成目标数字
  • 用户输入猜测
  • 根据结果提示大小
  • 循环直到猜中

面向对象初体验

Java最重要的思想之一就是面向对象。 通过 Class 和 Object， 我开始理解如何把现实中的事物抽象成代码。

例如： 游戏角色可以设计成一个对象： 拥有名字、等级、生命值等属性， 同时拥有攻击、防御等行为。

遇到的问题

刚开始学习 Java 时， 最大的困难是理解类、对象和方法之间的关系。 相比 C 语言， Java 更强调代码结构和模块化设计。

通过不断编写小程序， 我逐渐理解了面向对象编程的思想。

下一步计划

接下来我计划继续学习：

  • Java集合框架
  • 异常处理
  • 文件操作
  • Spring Boot基础
  • MySQL数据库开发

总结

Java学习第一周让我从简单语法进入了面向对象世界。 相比记忆知识点， 通过项目实践让我对软件开发有了更直观的认识。 未来希望能够使用 Java 开发完整的后端系统。
