☕ Java学习项目

项目介绍

Java 后端学习项目，涵盖 Java 基础语法、面向对象编程、Spring Boot 框架和 MySQL 数据库操作。通过实际项目练习，掌握企业级后端开发的核心技术。

学习内容

  • Java 基础语法与面向对象编程
  • Spring Boot 框架入门
  • RESTful API 设计与开发
  • MySQL 数据库设计与操作
  • MyBatis / JPA 数据访问

技术栈

Java Spring Boot MySQL

项目目标

通过系统学习 Java 后端技术栈，为后续参与实际企业项目开发打下坚实基础。
