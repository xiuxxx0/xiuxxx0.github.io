项目介绍

这是一个用于学习 Java 后端开发的实践项目。 目前正在从 Java 基础开始学习， 后续计划使用 Spring Boot 搭建完整后端系统。

学习目标

  • 掌握 Java 基础语法
  • 理解面向对象编程思想
  • 学习 Spring Boot 后端开发
  • 掌握 MySQL 数据库操作
  • 完成个人后端项目开发

技术栈

Java Spring Boot MySQL MyBatis Redis

计划实现功能

  • 👤 用户注册与登录
  • 🔐 JWT身份认证
  • 📝 博客管理系统
  • 📂 文件上传功能
  • 📊 数据统计接口

项目进度

目前阶段：

  • ✅ Java基础学习
  • ✅ 面向对象基础
  • 🔄 Spring Boot学习中
  • ⏳ 数据库设计
  • ⏳ 后端系统开发

开发收获

通过这个项目， 我逐渐从学习单个知识点， 转向理解真实软件项目的开发流程。

未来希望将这个项目完善为一个完整的后端应用， 并作为个人技术成长记录。
